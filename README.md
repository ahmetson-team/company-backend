# Company-Backend

## Before start
> [!IMPORTANT]
> 1. Run script to download and unzip PolygonID circuits.
> 2. Copy example of .env file and fill in your data.

```bash
./circuits.sh
cp .env.example .env
```

## How to start

To install dependencies

```bash
pnpm i 
```

To start API

```bash
pnpm start
```

> [!NOTE]
> All routes moved to PATH
> http://localhost:8000/api/v1/cashback/{...routes}
